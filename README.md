## Getting Started

Install the module with: `npm install mios`

```js
var mios = require('mios');
mios.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g mios
$ mios --help
$ mios --version
```


```sh
# creates a browser.js
$ grunt browserify
```
